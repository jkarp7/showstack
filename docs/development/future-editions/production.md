## Planned Features

### Production Scheduling (8–10 weeks)

- ⬜ Multi-view calendar (year, month, week, day, timeline, list)
- ⬜ Event types: rehearsal, performance, load-in, load-out, focus, tech, notes session
- ⬜ Drag-and-drop event creation and editing
- ⬜ Event templates (e.g., "Standard Tech Day", "Dress Rehearsal")
- ⬜ Recurring events with customizable patterns
- ⬜ Conflict detection (double-booked venue, crew, equipment)
- ⬜ Color-coding by department or event type
- ⬜ Export to iCal, Google Calendar, Outlook

### Budget Tracking (6–8 weeks)

- ⬜ Line-item budgets with categories (Labor, Equipment, Venue, Materials, Transportation, Other)
- ⬜ Estimated vs. actual tracking with variance analysis
- ⬜ Budget templates (save and reuse)
- ⬜ Department-level budgets and multi-project rollups
- ⬜ Export to Excel, PDF
- ⬜ Automatic cost calculation from crew bookings and shop order items
- ⬜ Equipment costs auto-populate from shop orders (rental/purchase/shipping rates)
- ⬜ Invoice tracking updates actual costs in real-time

### Labor & Crew Management (8–10 weeks)

- ⬜ Crew member profiles (contact info, emergency contacts, W-9 status)
- ⬜ Position management with pay rates and overtime multipliers
- ⬜ Crew booking system: drag-and-drop assignment, conflict detection, booking status workflow
- ⬜ Weekly hour tracking with overtime warnings
- ⬜ Payroll CSV export (ADP, Gusto, QuickBooks format)

### Inventory Management Extension (6–8 weeks)

- ⬜ Multi-department inventory (props, costumes, scenery, wardrobe, tools, consumables)
- ⬜ Ownership tracking (owned, rented, borrowed) with condition and location
- ⬜ QR code system — generate, print via label designer, check-out/check-in workflow
- ⬜ Rental period tracking with automatic cost calculation and late fees
- ⬜ Allocation conflict detection (item double-booked across shows)

### Vendor Management (6–8 weeks)

- ⬜ Vendor database (profiles, types, payment terms, tax ID, rating/review)
- ⬜ Purchase order system — create from shop order items, auto PO numbers, PDF export, status workflow
- ⬜ Invoice tracking — link to PO, payment status, overdue warnings, accounting export

### Tour Logistics (8–12 weeks)

- ⬜ Tour calendar — visual timeline, date types (load-in, focus, tech, performance, strike)
- ⬜ Venue database — technical specs, stage dimensions, loading dock, power, photo storage
- ⬜ Per diem calculator — standard rates by city/state, meal type tracking, currency conversion
- ⬜ Equipment manifests — packing lists, truck packing, check-in/check-out, pulls from existing inventory

### Production Books & Sharing (6–8 weeks, lower priority)

- ⬜ Production book builder — cover page, schedule, crew roster, venue info, attachments
- ⬜ Web sharing portal — public or password-protected, mobile-responsive, real-time updates, iCal embed
- ⬜ Access control — public link, password protection, expiration dates

### Reporting & Analytics (4–6 weeks, added incrementally)

- ⬜ Budget reports — summary, variance analysis, department rollup, vendor spending
- ⬜ Labor reports — crew hours, overtime, payroll export, cost by project
- ⬜ Schedule reports — production calendar PDF, crew call sheet, daily/weekly schedule, conflict report
- ⬜ Inventory reports — equipment allocation, rental costs, missing/damaged, maintenance schedule
- ⬜ Tour reports — schedule, venue contact list, per diem summary, equipment manifest, travel logistics
